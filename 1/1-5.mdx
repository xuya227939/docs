## 你将获得什么？

这不是一门教你"理论知识"的课程，而是一份**可执行的出海挣美金手册**。

学完这门课程，你将能够：

---

### 1. 拥有一个能赚钱的英文网站

不是停留在想法阶段，而是**真正上线、有流量、能变现**的网站。

- 完成从 0 到 1 的全流程
- 掌握网站搭建的技术栈
- 理解 SEO 和流量获取的底层逻辑

### 2. 建立可持续的被动收入

- 从 0 - 1 跑通挣美金的清晰路径
- 随着时间积累，收入会**复利增长**
- 不依赖单一平台，自己掌控流量和收入

### 3. 掌握一套可复制的方法论

学会方法后，你可以：

- 复制到不同领域，做多个站点
- 持续迭代优化，提升效率

---

## 具体技能

学完这门课程，你将掌握：

| 技能领域 | 你将学会 |
|----------|----------|
| **赛道选择** | 如何找到低竞争、高需求的细分市场 |
| **网站搭建** | 用 Next.js + Vercel 快速建站 |
| **内容创作** | AI 辅助写作 + 人工优化的高效流程 |
| **SEO 优化** | 关键词研究、内容优化、外链建设 |
| **流量获取** | 搜索引擎、社交媒体、社区推广 |
| **变现策略** | 广告、付费、订阅等多种变现方式 |
| **数据分析** | 用数据驱动决策，持续优化 |


## 适合谁学？

这门课程适合：

| 人群 | 为什么适合 |
|------|------------|
| **程序员/开发者** | 有技术背景，可以快速上手工具站 |
| **内容创作者** | 擅长写作，可以做博客站 |
| **产品经理** | 懂用户需求，能找到好的切入点 |
| **运营/市场人员** | 懂流量和推广，可以快速起量 |
| **想开启副业的上班族** | 利用业余时间，建立被动收入 |
| **自由职业者** | 增加收入来源，降低风险 |


## 需要投入多少时间？

| 阶段 | 时间投入 | 说明 |
|------|----------|------|
| 学习课程 | 1-2周 | 完整学完所有内容 |
| 搭建网站 | 1-2周 | 完成 MVP 上线 |
| 持续运营 | 每周 5-10 小时 | 内容更新、SEO 优化 |

**总计**：前期集中投入 4-6周时间，后期每周维护 5-10 小时。

> 💡 这是一个"前期投入、后期收获"的模式。前 3-6 个月投入时间，之后开始产生被动收入。

## 预期收益时间线

| 时间节点 | 预期成果 |
|----------|----------|
| 第 1 个月 | 完成网站搭建，上线 MVP |
| 第 2-3 个月 | 开始有少量流量（日均 10-100 UV） |
| 第 4-6 个月 | 流量增长，开始变现 |
| 第 6-12 个月 | 稳定增长 |
| 第 12 个月+ | 复制模式，多站点扩展 |

## 这门课程不适合谁

请注意，这门课程**不适合**：

- 想要"一夜暴富"的人
- 不愿意投入时间学习和执行的人
- 期望"躺赚"的人
- 没有耐心坚持 6 个月的人

这是一条需要**耐心和执行力**的路。如果你期望快速致富，这门课程不适合你。
